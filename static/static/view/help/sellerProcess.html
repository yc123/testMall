销售流程
1.上架产品
1.1 申请新品牌入库
第一步：在登入状态，进入卖家中心，点击【新品建档】，再点击【申请品牌】。
第二步：页面跳转至填写品牌基本信息及应用信息，填完并确认无误后，点击【确认无误，提交】。
第三步：页面自动跳转为我的品牌维护历史界面，请您耐心等待优软商城人员审核。
1.2 申请新标准元器件入库
若您需要的元器件原厂型号在优软商城没有找到，您可以申请标准器件入库。有则可省略该步骤。元器件申请流程如下：
第一步：在登入状态，进入卖家中心，点击【新品建档】，再点击【标准器件申请】
第二步：填写元器件基本信息
页面跳转至填写器件基本信息界面,请您填写”所属分类、”器件品牌”、”器件原厂型号”、”包装方式”等相关信息带*为必填。
第三步：填写元器件参数値
页面跳转至填写元器件参数値基本信息界面，完成元器件参数値，确认无误后点击【提交】。
1.3 标准器件批量申请
进入标准器件的批量申请的页面
1.点击批量申请标准器件
进入填写标准器件信息的页面
方法一：可以直接填写一级二级三级类目跟原厂官网产品目录下产品类表页网址，填完后，点直接提交按钮。
方法二：可直接下载表格按钮，把表格下载下来填写完后，点选择文件，选择刚填写好的表格，点击确认上传按钮即可。
1.4 批量发布产品
进入【批量发布库存页面】
1. 点击“下载Excel模板”。点击查看表格示例，可查看表格填写的格式，把表格信息补充完整，然后点击“上传库存信息”。
2. 上传完Excel表格后，点击“批量发布”，即可提示“发布成功”。
3.下载批量发布匹配不成功的型号列表。
上传过程中，与系统匹配不成功的型号会自动生产“匹配不成功的产品列表”，并且文件中包含验证不通过的原因，修改完删除错误信息再可重新上传。
2.发货
查看【卖家中心】的“销售订单管理”中的待发货状态，卖家可根据此状态下的订单进行发货。
